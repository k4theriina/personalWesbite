Projects

Since starting my CS journey, I’ve made an effort to getting comfortable with technology through creating various projects. Below are a list of them I’ve completed, which can all be found on my GitHub!

Rock, Paper, AI Sensors

STEM Day | August - October 2024

This was a program designed for kids to learn about AI through playing a “Rock, Paper, Scissors” game against the computer. Using AI, the program would recognize the players’ hands and give a random output from rock, paper, or scissors. I designed and created the front-end of this program using TKinter. My team and I further collaborated on testing the program and finally presented this to 70+ kids.

Bloom Buddy

Knight Hacks VII | October 2024

During UCF’s 2024 Hackathon, KnightHacks VII, my team and I created “Bloom Buddy”, a mobile app created with Flutter that displays a plant’s information such as its moisture, temperature, and brightness. We utilized hardware to measure these statistics and make the users plant’s self-watering. I worked on designing the logo and look of the app as well as programming the front-end.

To-do List Website

Personal Project | September 2024

This was my first project, where I introduced myself to the world of web development. Using HTML, CSS, and JavaScript, I created a to-do list website that allows the user to create new tasks, mark tasks as completed, and delete tasks. I personally use this website frequently, as it serves as a quick and easy organizer for my everyday tasks.
